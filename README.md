# 🛒 Blinkit Grocery Sales Analysis Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Power BI dashboard** developed using the **Blinkit Grocery Sales** dataset. The objective was to transform raw sales data into meaningful business insights through data cleaning, modeling, DAX calculations, and visualization.

The dashboard enables users to analyze sales performance, outlet efficiency, customer satisfaction, and product performance through interactive reports and KPIs.

---

## 🚀 Project Objectives

- Clean and transform raw data using Power Query.
- Build an efficient data model.
- Create business KPIs using DAX.
- Design an interactive dashboard for business analysis.
- Generate actionable insights from sales and customer data.

## 📈 Key Business Insights

### 🥇 1. Top Selling Product Category
- **Fruits & Vegetables** contribute the highest share of total sales, making them the best-performing product category.

### 🥛 2. Fat Content Analysis
- Products with **Regular Fat** generate higher sales than **Low Fat** products.

### 🏪 3. Outlet Performance
- **Tier 3 outlets** outperform Tier 1 and Tier 2 outlets in overall sales.
- **Small-sized outlets** achieve the highest sales among all outlet sizes.

### 🛍️ 4. Top Performing Product Categories
The following product categories consistently perform well based on sales and customer ratings:

- Baking Goods
- Breads
- Breakfast
- Canned Foods
- Dairy
- Frozen Foods
- Fruits & Vegetables
- Hard Drinks
- Household
- Hygiene
- Meat
- Others
- Seafood
- Snack Foods

### ⭐ 5. Customer Satisfaction
- There is minimal variation in sales across different outlet types.
- Customer satisfaction varies significantly across product categories, indicating that item type has a greater influence on customer ratings than outlet type.
